Hugo搭建个人博客

如何使用hugo搭建个人博客 本机操作（windows操作系统） 1.下载安装包 https://github.com/gohugoio/hugo/releases 2.解压，放在：E:\Software\hugo 目录下 3.将 E:\Software\hugo 放到PATH中 4.重启cmder,查看hugo版本 5.去hugo官网点击quick start,按照文档操作 6.找一个合适的文件夹,用于创建博客的生成器目录 7.在此文件夹下，打开终端，执行： hugo new site username.github.io-creator ps：username代表自己的github用户名 8.创建主题 cd进入创建好的username.github.io-creator目录下，执行 git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo 'theme = "ananke"' >> config.toml 9.添加内容 终端输入 hugo new posts/我的第一篇博客.md 10.在vscode下打开username.github.io-creator目录,可以在content目录下看到我的第一篇博客.md 11.写一些内容之后，终端执行 hugo server -D 得到url来预览当前博客 12.预览无误就将draft: true状态修改为false,为上传做准备 13.在username.github.io-creator目录下，打开config.toml文件，修改 baseURL = "http://username.github.io/" languageCode = "zh-Hans" ps：username代表自己的github用户名 14.终端运行hugo hugo连接到github上 1.进入username.github.io-creator目录，新建.gitignore文件，写入 /public/ 2.进入public目录,再建一个本地仓库专门管理public目录 git init 3.然后 git add. git commit -v GitHub操作 1.
